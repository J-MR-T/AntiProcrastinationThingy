# AntiProcrastinationThingy

Just a personal tool to help me stay focused and not open apps I don't want to open when concentrating or working on a
thing

## Commandline Arguments:
- -volume=...
- -w/-width=...
- -h/-height=...
- -theme=awesome/dark/light
